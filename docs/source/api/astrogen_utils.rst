astrogen_utils module
======================

This module contains useful functions to manipulate strings for name matching,
pretty printing, gender detection and XLSX output.

The functions in this module comprise:

+ string comparisson:
    + ds
    + ds1
    + ds2
    + initials
    + getinitials
    + pickone

+ input/output:
    + bcolors
    + append_df_to_excel
